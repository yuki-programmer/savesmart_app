# 実践例

アプリ種別ごとの心理学的手法の活用例。

## 家計簿・節約アプリ

### SaveSmart型アプローチ

**メンタルアカウンティング活用**
- 3カテゴリ（貯蓄/通常/ごほうび）で心理的区分
- 「今日使える金額」で日次の意思決定を簡略化

**選択のパラドックス回避**
- カテゴリを3つに限定（多すぎない）
- 複雑な予算設定不要

**即時フィードバック**
- 支出入力直後に残額更新
- 目標達成時のアニメーション

**損失回避**
- 「あと○円使うと貯蓄目標に届かなくなります」
- 赤字時のビジュアル警告

**コミットメント**
- 月初の貯蓄目標設定
- 達成率の可視化

## ECアプリ

**希少性**
- 「残り2点」表示
- 「○人がカートに入れています」

**社会的証明**
- レビュー数・星評価
- 「この商品を買った人はこれも」

**アンカリング**
- 元値と割引価格の併記
- 「○%OFF」の強調

**損失回避**
- 「カートの商品、まもなく売り切れ」
- 期間限定セール

**サンクコスト**
- お気に入り登録済み商品のリマインド

## タスク管理・生産性アプリ

**目標勾配効果**
- タスク完了プログレスバー
- 「今日のタスク: 3/5完了」

**ツァイガルニク効果**
- 未完了タスクのバッジ表示
- 「中断したタスクがあります」通知

**可変報酬**
- 連続達成でランダムバッジ
- 週間サマリーでの称賛

**実装意図**
- 「いつやるか」のスケジュール機能
- リマインダーの具体的設定

## SNS・コミュニティアプリ

- 通知のランダム性

**バンドワゴン効果**
- トレンドトピック

## サブスクリプションサービス

**デフォルト効果**
- 推奨プランをデフォルト選択
- 年間プランの「お得」表示

**フレーミング**
- 「月額換算○円」（年払いを安く見せる）
- 「1日あたり○円」

**損失回避**
- 解約前に「これがなくなります」表示
- トライアル終了リマインド

**サンクコスト**
- 「○ヶ月間ご利用いただいています」
- 蓄積データの表示
